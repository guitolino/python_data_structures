# Python Data Structures

A lightweight Python package providing ready-to-use data structures (Stack, LinkedList, StrictStack, etc.).

## How to use

1. Clone the repo
>Via HTTPS:
```bash
git clone https://github.com/guitolino/python_data_structures.git
```
---
>Via SSH:

```bash
git clone git@github.com:guitolino/python_data_structures.git
```
---
>Via GitHub CLI:

```bash
gh repo clone guitolino/python_data_structures
```

2. Install dependencies:
```bash
#from project root, execute:
pip install -r requirements.txt
```

3. Install the module:
```bash
#from project root, execute:
pip install -e .
```

## Running tests

```bash
#from project root, execute:
pytest
```
